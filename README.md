# TOPIQ — 韩语背单词 PWA

基于 SM-2 间隔重复算法的 TOPIK 韩语词汇学习应用。移动端优先，纯静态站点，GitHub Pages 一键部署。

**1980 词 · 40 单元 · 初级+中级 · 例句+发音+统计**

---

## 功能

### 核心学习
- **SM-2 间隔重复**：忘记/模糊/认识 三档评分，算法自动计算下次复习时间
- **单词卡片**：CSS 3D 翻转，正面韩语 → 背面释义+例句+词源注释
- **撤销评分**：按错可撤回，3 秒内恢复
- **学习范围**：全部 / 仅初级 / 仅中级，随时切换

### 单词库
- **40 单元浏览**：按书本单元查看，每单元显示学习进度条
- **快速复习**：单元内逐词卡片浏览，或打乱全部单词随机穿插
- **收藏系统**：星标难词，筛选集中攻克
- **全文搜索**：支持韩语、中文、例句内容检索

### 发音
- **智能语音**：自动选用系统最佳韩语语音包（Google / Microsoft Heami）
- **语音选择器**：列出可用语音，点击试听
- **单词 + 例句朗读**：卡片正面单词、背面例句均可一键发音
- **语速可调**：0.5x ~ 2.0x

### 统计
- **遗忘曲线**：基于实际复习数据的记忆保持率
- **学习情况**：近 30 天复习 + 未来 7 天预测
- **掌握分布**：Lv0 ~ Lv7+ 各等级词数
- **签到日历**：连续打卡天数

### 数据管理
- **导出/导入**：JSON 备份学习进度、签到、助记、收藏
- **重置进度**：两重确认防误操作

### 体验
- **深色模式**：自动跟随或手动切换
- **PWA 支持**：添加到 iPhone/Android 主屏幕，全屏体验
- **键盘快捷键**：1/2/3 评分，Space 翻转，Enter 保存

## 技术栈

| 层 | 技术 |
|----|------|
| 框架 | 无框架，Vanilla JavaScript |
| 算法 | SM-2 (SuperMemo 2) 客户端实现 |
| 持久化 | localStorage |
| 图表 | Chart.js 4.x |
| 语音 | Web Speech API (TTS) |
| 样式 | 纯 CSS，移动端优先响应式 |
| 部署 | GitHub Pages，纯静态 |

项目无构建工具、无 npm 依赖（仅 CDN 引入 Chart.js），直接打开 `index.html` 即可运行。

## 快速开始

### 本地运行

```bash
# 任意 HTTP 服务器均可
npx serve .
# 或
python -m http.server 8080
```

然后浏览器打开 `http://localhost:8080`。

### GitHub Pages 部署

1. Push 到 GitHub 仓库
2. Settings → Pages → Source: `main` branch, root (`/`)
3. 保存，等 1-2 分钟即可访问 `https://<username>.github.io/<repo>/`

## SM-2 算法说明

三个评分等级映射到 SM-2 的 q 值：

| 按钮 | q 值 | 效果 |
|------|------|------|
| 忘记 | 0 | 级别重置为 0，当天重新学，ease factor -0.2 |
| 模糊 | 3 | 级别前进，间隔减少，ease factor -0.15 |
| 认识 | 5 | 级别前进，间隔指数增长，ease factor +0.1 |

级别与间隔示例（全"认识"路径）：

```
Lv0 → Lv1: 1 天
Lv1 → Lv2: 1 天
Lv2 → Lv3: 3 天
Lv3 → Lv4: 8 天
Lv4 → Lv5: 20 天
```
